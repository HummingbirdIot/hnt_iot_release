### Speed up github access

For User `pi`
```bash
git config --global url."https://github.com.cnpmjs.org/".insteadOf "https://github.com/"
```

For User `root`
```bash
sudo su
git config --global url."https://github.com.cnpmjs.org/".insteadOf "https://github.com/"
```
